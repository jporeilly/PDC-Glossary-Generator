# Module 02 — Build Your Own AWC Dictionary & Pattern (Lab)

**Track:** Technical · **Audience:** Data Steward / Data Storage Administrator
**Estimated time:** 45 min · **Prerequisite:** profiled AWC tables; Module 01

A hands-on lab that extends Module 01: you author **one dictionary** and **one
pattern**, combine them into a **policy**, and run it on AWC's data.

- **`Lab-Build-Your-Own-AWC-Dictionary-and-Pattern-Deck.pptx`** — the walkthrough deck
  (Part A dictionary, Part B pattern, Part C run, what you should see, wrap-up).
- **`Lab-Build-Your-Own-AWC-Dictionary-and-Pattern.docx`** — the full standalone lab guide.

Worked examples: an **AWC Water-System Codes** dictionary (match by content) and an
**AWC Account Number** pattern `AAA-nnnnnnn` (match by shape). Profile first; author
once; the Trust Score comes last. Next module: **03 — Glossary Generator App**.

All data is fictional and generated for training.
