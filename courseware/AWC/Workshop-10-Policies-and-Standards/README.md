# Workshop 10: Govern with Policies & Standards

**Primary role:** Business Steward  
**Estimated time:** 60 min

## What's in this package

- **`Workshop-10-Policies-and-Standards.pptx`** — the slide deck for this session (with speaker notes)
- **`Workshop-10-Guide.docx`** — the detailed standalone workshop guide. Read this; it explains the steps *and the reasoning*, with Arizona Water Company context.

## Assets used in this workshop

- `assets/Arizona-Water-Company-Business-Rules.sql`
- `assets/Arizona-Water-Company-Glossary.csv`

## How to run it

1. Make sure the AWC lab is running (`make all` in the awc-lab bundle).
2. Make sure you have completed the previous workshops — this one builds on them.
3. Present the deck, then follow the guide step by step.

All data is fictional and generated for training.
