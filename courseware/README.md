# Courseware — two scenarios, two complete sets

Each scenario is self-contained. Use one set end-to-end; don't mix them.

| Set | Scenario | Data sources kit | Domain pack |
| --- | --- | --- | --- |
| [`AWC/`](AWC/) | **Arizona Water Company** — water utility (the original 1.6.x workshop) | `data_sources/AWC/` | `data_sources/AWC/awc-domain-pack.zip` |
| [`CSCU/`](CSCU/) | **Copper State Credit Union** — financial services (current) | `data_sources/CSCU/` | `data_sources/CSCU/cscu-domain-pack.zip` |

The **AWC set** is the complete original course: Workshops 00–11 (decks +
Word guides + assets), the Technical Track, and the Glossary Generator
workshop document and deck.

The **CSCU set** currently covers **Workshops 00–05** (Preflight → Data
Identification): per-workshop READMEs, markdown guide masters with
`[SCREENSHOT]` markers, and all scenario assets (datasources CSV, documents,
users, generated glossary + term-linking map, business rules, dictionaries).
Decks and .docx builds for CSCU are pending — generate them from the guides.
Workshops 06–11 have not been produced for CSCU yet.

*All scenario data is fictional and generated for training.*
