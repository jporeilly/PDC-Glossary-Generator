# Module 05 — Visualizations (PDC Insights)

**Track:** Technical · **Audience:** Solution Architect / Data Developer

**PDC Insights** (`pdc-insights-1.3.0.zip`) — a separate, scenario-neutral
companion app that polls the Glossary Generator's read-only
`GET /api/governance-summary` endpoint (vocabulary health, steward audit
summary, drift) and renders dashboards; every dashboard can be downloaded as a
spec and printed to PDF. See its bundled `CHANGELOG.md`; `print-app.png` shows
the print view.

Run it beside the Glossary Generator (it defaults to port 8660 — keep it apart
from the app's 5000).

All Copper State Credit Union data is fictional and generated for training.
