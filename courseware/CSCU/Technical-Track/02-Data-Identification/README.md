# Module 01 — Data Identification

**Track:** Technical · **Audience:** Data Steward / Data Developer / Admin
**Pairs with:** BA Workshop 5 — Protect Sensitive Data

The conceptual deep-dive into PDC's auto-classification engine — how **Dictionaries**
(match by content) and **Data Patterns** (match by shape) combine into **Policies**
that apply **Tags and Business Terms** to profiled data.

- **`Data-Identification-Technical.pptx`** — the session deck (the two methods,
  confidence rules, position analysis, policies, running a job, a worked AWC PII policy).
- **`Data-Identification-Guide.docx`** — the standalone technical guide.

Runs on **profiled** data; the Trust Score is computed **last**. Next module:
**02 — Build Your Own AWC Dictionary & Pattern** (hands-on).

All data is fictional and generated for training.
