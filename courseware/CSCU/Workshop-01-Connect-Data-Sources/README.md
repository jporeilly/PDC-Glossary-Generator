# Workshop 1: Connect the Data Sources

**Primary role:** Data Steward / IT Administrator
**Estimated time:** 120 min

## What's in this package

- **`Workshop-01-Guide.md`** — the workshop guide (markdown master, CSCU context).

> The deck and Word guide for the CSCU edition have not been produced yet —
> `Workshop-01-Guide.md` is the authoritative source: build the .docx/.pptx from it
> and capture the screenshots at each `[SCREENSHOT]` marker on the CSCU lab.

## Assets used in this workshop

- `assets/cscu-datasources.csv`
- `assets/cscu-documents/  (the full 18-file bucket set — compliance PDFs, correspondence, loan applications, statements, payments, rates)`

## How to run it

1. Make sure the shared lab is running with CSCU loaded
   (`data_sources/lab/`: `make up && make load SCENARIO=CSCU`).
2. Make sure you have completed the previous workshops — this one builds on them.
3. Work through `Workshop-01-Guide.md` step by step.

All Copper State Credit Union data is fictional and generated for training.
