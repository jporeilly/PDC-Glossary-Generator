# Workshop 3: Build the Business Glossary

**Primary role:** Business Steward / Business Analyst
**Estimated time:** 45 min

## What's in this package

- **`Workshop-03-Guide.md`** — the workshop guide (markdown master, CSCU context).

> The Word guide (`.docx`) is **generated from the markdown master** — the
> `.md` stays authoritative. Amber boxes in the .docx mark where to paste
> screenshots captured on the CSCU lab. Regenerate after editing the master
> with `courseware/CSCU/tools/build-docx.py`. The deck (.pptx) is pending.

## Assets used in this workshop

- `assets/CSCU-Business-Glossary.jsonl  (import-ready glossary — generated by the Glossary Generator pipeline from the cscu_core schema)`
- `assets/CSCU-Term-Linking-Map.csv  (column → business-term links)`
- `assets/cscu_metadata_dictionary.csv  (schema/table/column/type/comment reference)`
- `assets/CSCU-glossary-user-map.csv  (steward name → roster UUID)`
- `assets/users.csv`

## How to run it

1. Make sure the shared lab is running with CSCU loaded
   (`data_sources/lab/`: `make up && make load SCENARIO=CSCU`).
2. Make sure you have completed the previous workshops — this one builds on them.
3. Work through `Workshop-03-Guide.md` step by step.

All Copper State Credit Union data is fictional and generated for training.
